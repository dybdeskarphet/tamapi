Overview

This API provides resources to manage and interact with virtual pets and related user data.

Endpoints

Auth Endpoints

  • POST /auth/register - Register a new user
  • POST /auth/login - Get a token by logging in
  • POST /auth/test-user - Register/login as a test-user (dev-only)

Pet Endpoints

  • GET /pets - Get all pets of the user (user token required)
  • GET /pets/:id - Get a pet by its ID (user token required)
  • POST /pets - Create a new pet (user token required)
  • POST /pets/:id/history - Get the action history of a pet (user token required)
  • DELETE /pets/:id - Delete a pet (user token required)
  • PATCH /pets/:id - Patch available information about the pet (user token required)
  • POST /pets/:id/:action - Perform an action on pet (user token required)

Pet Actions

  • feed - Feed the pet.
  • play - Play with the pet.
  • sleep - Put the pet to sleep.
  • clean - Clean the pet.
  • heal - Heal the pet.

      • Profile Endpoints

      • GET /profile - Get the current information about the user (user token required)
